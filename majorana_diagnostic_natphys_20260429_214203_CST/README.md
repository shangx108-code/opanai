This is the long-term project space for the Majorana diagnostic manuscript reboot. From this point onward, data, derivations, manuscript assets, code, and run logs for this chat should be stored here by default.

## Directory structure

- `data/raw/`: raw regenerated numerical outputs and imported source tables
- `data/processed/`: merged tables, summaries, and analysis-ready outputs
- `figures/`: regenerated figure assets and panel-level provenance
- `manuscript/`: draft text, section packages, bibliography, and submission files
- `derivations/`: derivation notes, notation ledgers, and theory blocks
- `code/`: runnable scripts and environment helpers
- `logs/`: run logs, environment checks, and iteration checkpoints
