# self-calibrating-diffractive-ncomms manuscript package

This directory contains a Nature Communications style manuscript package built
from the current verified and partially verified ledger for the project
`self-calibrating-diffractive-ncomms`.

## Current package contents

- `manuscript/main.tex` and `manuscript/main.pdf`: main manuscript
- `supplement/supplement.tex` and `supplement/supplement.pdf`: Supplementary Information
- `figures/`: Figure 1 schematic and Figures 2-4 data figures
- `data/`: processed figure tables and method fairness table
- `source_data/source_data_index.csv`: figure-to-data traceability table
- `scripts/render_figures.py`: figure rendering script for the data figures

## Evidence boundary

- Figure 1 is a schematic generated for scientific communication.
- Figures 2-4 are rendered only from already reported project values preserved in
  the project memory and uploaded materials.
- The current package is submission-style and evidence bounded.
- The processor-level claim remains intentionally limited.

## Archive note

The historical round1-round6 raw archive is not yet fully restored into this
active package. The current directory therefore captures the manuscript assembly
state and the processed figure tables used for the present draft, not a full
reproduction bundle.
