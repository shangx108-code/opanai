# Minimal two-scale calibration and interface frustration screening for superconducting order in twisted bilayer WSe2

## Abstract

Identifying the superconducting order parameter in twisted bilayer WSe2 requires a modeling pipeline that can distinguish robust pairing fingerprints before a full transport-level calculation is completed. We present a minimum runnable theory engine that already separates two tasks that are often entangled in broader numerical studies: calibration of the normal and Bogoliubov spectral scales, and rapid screening of candidate gap structures at an interface proxy. Within the present parameter set, the normal-state density of states yields an outer positive-energy scale of 0.674, while the Bogoliubov spectrum yields an inner scale of 0.118, giving a scale ratio of 5.71. On top of this calibrated background, the interface screening cleanly distinguishes three representative pairing families. A conventional `s`-wave state remains frustration-free for all intervalley-mixing weights considered here, an `s_\pm`-like state is activated only when intervalley reflection is introduced, and a chiral phase-winding state retains a finite frustration signal even without intervalley conversion. These results do not yet constitute a PRL-complete conductance calculation, but they establish a compact and executable preselection layer that narrows which order-parameter families warrant full Andreev or BTK follow-up.

## Main text

Twisted transition-metal dichalcogenide bilayers provide a setting in which superconductivity, spin-valley locking, and symmetry lowering can coexist on comparable energy scales [citation needed]. For twisted bilayer WSe2 in particular, the central theoretical difficulty is not only to propose candidate order parameters, but to decide which signatures are robust enough to survive the translation from band structure to interface response [citation needed]. A useful short-format theory result, especially at the present exploratory stage, is therefore not a broad catalogue of possibilities. It is a compact executable filter that identifies which pairing classes already separate at the level of symmetry-constrained interface frustration and which ones still require heavier transport numerics.

The present work isolates that filtering step. We start from a minimal valley-spin normal Hamiltonian with nearest- and next-nearest-neighbor hopping, a valley texture term, a displacement-field-like contribution, and a correlated background scale `\Delta_{\rm corr}`. Superconductivity is then introduced in three representative channels: a conventional `s`-wave state, a valley-odd `s_\pm`-like state, and a chiral phase-winding state. The purpose of this engine is deliberately narrow. It is not yet to reproduce a final conductance line shape. It is to test whether the existing parameterization already contains a stable two-scale spectral calibration and whether a simple interface proxy can discriminate qualitatively distinct pairing families before a full PRL-grade BTK computation is built.

The first output is a two-scale calibration from the density of states. Using the current parameter set (`\mu=-1.65`, `t_1=1.0`, `t_2=0.18`, `v_1=0.32`, `m_z=0.22`, `\Delta_{\rm corr}=0.34`, `\Delta_{\rm sc}=0.11`), the positive-energy normal-state density of states develops peaks at 0.568 and 0.674, while the Bogoliubov spectrum shows positive-energy peaks at 0.118, 0.250, and 0.572. We use the largest normal-state peak to define an outer spectral scale and the smallest Bogoliubov peak to define an inner superconducting scale. This yields `E_{\rm out}=0.674`, `E_{\rm in}=0.118`, and `E_{\rm out}/E_{\rm in}=5.71`. In the present manuscript these numbers should be read as a calibration anchor, not yet as a direct fit to experiment. Their immediate value is that they provide an internally consistent hierarchy on which later interface calculations can be normalized.

The second output is an interface-frustration screening metric. For a boundary orientation `\alpha` and intervalley-mixing weight `\eta`, we compare the gap phase experienced by incoming and reflected trajectories, including both same-valley and intervalley reflection channels. The resulting scalar frustration measure is averaged over incidence angle and scanned across candidate order parameters. This construction is intentionally lightweight, but it is sufficient to expose whether a candidate is quiet, intervalley-activated, or intrinsically phase-frustrated at the boundary.

Three distinct behaviors emerge. First, the conventional `s`-wave state is featureless in this metric: the averaged frustration remains numerically at zero for all `\eta` between 0 and 1. Second, the `s_\pm`-like state is also quiet when intervalley reflection is absent, but rises monotonically with intervalley mixing, reaching an averaged frustration of essentially 1 at `\eta=1`. This is the clearest signature in the present minimal engine that a valley-odd sign structure can remain hidden in purely intravalley reflection yet become sharply exposed once interface scattering mixes valleys. Third, the chiral phase-winding state shows a qualitatively different pattern: its averaged frustration is already finite, 0.533, at `\eta=0`, and remains at that value across the full intervalley scan considered here. In other words, the chiral candidate carries a built-in boundary frustration that does not rely on intervalley conversion.

These three outcomes already define a useful decision tree for subsequent PRL-focused numerics. If future conductance calculations are meant to test a conventional `s`-wave scenario, the present proxy predicts no interface-frustration-driven enhancement at all. If the target is a valley-odd `s_\pm`-type state, the decisive regime is not generic interface scattering but the intervalley-active limit. If the target is a chiral state, the relevant question is instead whether a finite zero-intervalley frustration signal survives after the proxy is replaced by a full transport kernel. The main gain is therefore not merely classificatory. It is computational: one can prioritize different BTK extensions for different pairing families instead of treating all candidates with the same expensive workflow.

At the same time, the present evidence boundary should be stated plainly. This engine does not yet compute an experimentally resolved Andreev conductance spectrum, does not include a self-consistent interface potential, and does not by itself establish which pairing state is realized in twisted bilayer WSe2. The current results support a narrower claim: a minimal two-step pipeline already exists in executable form, and within that pipeline the three representative pairing families separate in ways that are physically interpretable and operationally useful. For a PRL submission, the next non-negotiable step is to transplant these distinctions into a full transport calculation and show that at least one of the screening branches remains visible in a measurable observable.

Within that boundary, the present minimal engine offers a concrete starting point for a sharper PRL story. The spectral calibration fixes a nontrivial inner-to-outer hierarchy, and the boundary proxy shows that intervalley-activated and intrinsically phase-winding pairing families need not be degenerate at the interface level. What has been achieved here is therefore not a complete superconducting diagnosis, but a compact preselection framework that meaningfully reduces the space of viable follow-up calculations.

## Data-backed statements used in this draft

- Normal-state positive-energy peaks: `(0.568, 0.4645)` and `(0.674, 0.4713)`.
- Bogoliubov positive-energy peaks: `(0.118, 1.0965)`, `(0.250, 0.8562)`, and `(0.572, 0.8248)`.
- Outer scale: `0.674`.
- Inner scale: `0.118`.
- Outer/inner ratio: `5.71`.
- `s`-wave frustration remains numerically near zero for `\eta=0, 0.25, 0.5, 0.75, 1.0`.
- `s_\pm` frustration grows from approximately `0` at `\eta=0` to approximately `1` at `\eta=1`.
- Chiral frustration remains approximately `0.533` for the full `\eta` scan.
