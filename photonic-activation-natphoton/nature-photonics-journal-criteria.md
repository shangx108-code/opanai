# Nature Photonics Journal Criteria

## Target journal
- Nature Photonics

## Criteria that remain binding
1. The main claim must matter to photonics readers beyond one machine-learning demo.
2. Any "quantum limit" language must be tied to explicit assumptions and an explicit optimization target.
3. Device consequences must be visible, not buried inside abstract metrics.
4. The paper must say where the proposed route loses, not only where it wins.
5. Recent 2024-2025 photonic nonlinearity papers remain the direct novelty pressure point.

## Criteria status after this run
- Scope fit: still good
- Conceptual sharpness: improved again because one lower bound is now connected to one concrete measurement route
- Theory completeness: still incomplete
- Numerical evidence completeness: improved from initial to first same-axis comparison, but still insufficient
- Device relevance: improved to partial because detector efficiency now produces a direct penalty map
- Broad-reader accessibility: still underdeveloped
- Literature separation: still insufficient

## What this run newly satisfies
- The project now has one concrete measurement-induced activation route on the same axes as the lower bound.
- The project can now state a bounded device-facing message:
  homodyne-conditioned activation stays near the frontier only when detector efficiency is very high.

## What this run still does not satisfy
- No full paper-level limit theorem
- No second architecture curve yet shows whether the observed behavior is distinctive or generic
- No submission-grade figure generated from the new result
- No direct positioning text yet against the most relevant recent papers

## Current venue risk summary
- The paper is still at risk of being seen as an interesting but preliminary framework unless the new same-axis comparison is expanded into either a multi-route benchmark or a figure/manuscript package strong enough to feel definitive.
