# Nature Photonics Journal Criteria

## Target journal
- Nature Photonics

## Criteria that remain binding
1. The main claim must matter to photonics readers beyond one machine-learning demo.
2. Any "quantum limit" language must be tied to explicit assumptions and an explicit optimization target.
3. Device consequences must be visible, not buried inside abstract metrics.
4. The paper must say where the proposed route loses, not only where it wins.
5. Recent 2024-2025 photonic nonlinearity papers remain the direct novelty pressure point.

## Criteria status after this run
- Scope fit: still good
- Conceptual sharpness: improved slightly because one activation-cost primitive is now operationally defined
- Theory completeness: still incomplete
- Numerical evidence completeness: improved from absent to initial but still insufficient
- Device relevance: still unproven
- Broad-reader accessibility: still underdeveloped
- Literature separation: still insufficient

## What this run newly satisfies
- The project now has one explicit, assumption-bounded lower-bound statement that is compatible with the journal's demand for interpretable physics rather than rhetorical generalities.
- The project now also has initial numerical evidence showing how the lower-bound cost grows as the allowed boundary error shrinks, which is closer to the journal's expectation of design-relevant consequence.

## What this run still does not satisfy
- No full paper-level limit theorem
- No concrete device-architecture consequence shown from comparison on the same axes
- No submission-grade figure generated from the new result
- No direct positioning text yet against the most relevant recent papers

## Current venue risk summary
- The paper is still at risk of being seen as an interesting perspective-level framework rather than an Article unless the new lower-bound numerics are connected to a concrete measurement route and a device-relevant comparison.
