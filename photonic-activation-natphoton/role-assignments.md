# Role Assignments

## Coordinator
- Current task: convert the new lower-bound-versus-homodyne comparison into a figure-grade result package and choose the next same-axis baseline
- Inputs: theory framework, homodyne comparison note, venue criteria, manuscript spine
- Deliverable: execution brief for the second comparison cycle and Figure-3 candidate scope
- Completion standard: the next run can immediately compute the next baseline and assemble one submission-grade panel
- Dependency: none

## Theory
- Current task: derive the cleanest next comparison route, most likely direct counting or displaced counting, on the same discrimination-cost axes
- Inputs: lower-bound derivation, homodyne comparison note, theory framework
- Deliverable: explicit error model and assumptions for the second baseline
- Completion standard: the next baseline can be computed without ambiguity and without overclaiming
- Dependency: coordinator framing

## Code and Numerical Computation

## Data Analysis

- Deliverable: interpretation memo for what counts as near-frontier, degraded, and not-worth-it operation

- Dependency: theory + computation plan
