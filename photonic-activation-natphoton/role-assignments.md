# Role Assignments

## Coordinator
- Current task: convert the formalized framework into the first executable numerical cycle
- Inputs: theory framework, benchmark spec, manuscript spine, venue criteria
- Deliverable: round-2 execution brief with the first numerical milestone
- Completion standard: the next run can immediately start implementing or specifying result-producing calculations
- Dependency: none

## Theory
- Current task: sharpen the framework into theorem-like or proposition-like statements about regime boundaries
- Inputs: theory framework, literature pressure points
- Deliverable: a concise proposition set or protocol statement for the manuscript
- Completion standard: the paper can name exactly what is being proved, bounded, or compared
- Dependency: coordinator framing

## Code and Numerical Computation
- Current task: instantiate the first benchmark from the specification, starting with single-neuron response and cost curves
- Inputs: benchmark specification, theory definitions
- Deliverable: run-ready calculation checklist and initial implementation order
- Completion standard: the first numerical cycle can produce Figure-3-style outputs
- Dependency: theory definitions

## Data Analysis
- Current task: translate benchmark outputs into acceptance-relevant decision criteria
- Inputs: benchmark specification, manuscript spine
- Deliverable: analysis rubric for judging regime boundaries and superiority windows
- Completion standard: no forthcoming figure can be misread as merely qualitative
- Dependency: theory + computation plan

## Figure Production
- Current task: align Figure 2-6 with the new theory framework and benchmark outputs
- Inputs: theory framework, benchmark spec, manuscript spine
- Deliverable: revised storyboard tied to concrete numerical deliverables
- Completion standard: figure order now matches the manuscript proof logic
- Dependency: coordinator + data-analysis memo

## Writing
- Current task: draft the first manuscript package around the new spine
- Inputs: manuscript spine, theory framework, figure storyboard
- Deliverable: title, abstract, introduction logic, results headings, and Methods skeleton
- Completion standard: a draft can be assembled immediately once the first numerical figures exist
- Dependency: coordinator + theory + figure plan

## Supervision
- Current task: keep the project from drifting into an under-supported manifesto
- Inputs: central claim, venue criteria, literature map
- Deliverable: supervision note naming the top three rejection risks and the corresponding evidence demands
- Completion standard: no claim of "quantum advantage", "universal", or "near-optimal" remains unsupported
- Dependency: coordinator output

## Strict Review
- Current task: update the five-reviewer baseline after formalization
- Inputs: theory framework, benchmark spec, manuscript spine
- Deliverable: revised reviewer matrix and next upgrade conditions
- Completion standard: acceptance probability tracks actual project maturation rather than the original concept note only
- Dependency: coordinator output
