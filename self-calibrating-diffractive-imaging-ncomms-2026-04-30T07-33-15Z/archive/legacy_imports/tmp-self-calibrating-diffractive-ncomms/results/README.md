# Stable Results Landing Zone

This directory is the stable results entrypoint for `self-calibrating-diffractive-ncomms`.

## Rule

- Put machine-readable computation outputs here first.
- Keep one subdirectory per round or per named evidence package.
- Prefer `.csv`, `.json`, and `.md` for the first sync step.
- Keep binary figure panels in archive snapshots until a stable figure-pack convention is locked.

## Current coverage

- `round6/`: first synchronized minimal NumPy passive D2NN rebuild package

## Minimum upload set for each new round

1. metrics table
2. summary JSON
3. summary Markdown
4. environment note if the runtime chain changed
