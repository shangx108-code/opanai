# Code Availability

The code used to generate the restored round6 processor-level baseline is included in this project space under `code/project-scripts/round6_numpy_passive_d2nn.py`. The script that regenerates the manuscript-facing source-data and registry files is provided as `code/project-scripts/refresh_submission_package.py`. Additional project code will continue to be archived in this GitHub-backed long-term project space and can be deposited to a public archival repository before submission if required by the journal.
