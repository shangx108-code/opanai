# Journal-format micro-adjustment plan: Nature Physics and PRX

## Nature Physics-facing version
- Keep the central contribution line concise: field-free topological superconductivity plus a false-positive-resistant nonlocal diagnostic.
- Use a compact main text with schematic motivation, bulk transition, boundary evidence, negative controls and nonlocal transport.
- Keep technical derivations, threshold definitions, ensemble details and reproducibility notes in Supplementary Information.
- Avoid overclaiming Majorana detection; use calibrated language such as "identifies topological superconductivity" and "false-positive-resistant diagnostic".

## PRX-facing version
- Emphasize methodological generality: a combined bulk-boundary-transport diagnostic for phase-engineered topological superconductivity.
- Retain more explicit definitions in Methods, including the BdG Hamiltonian, class-D invariant, and transport score.
- Keep the statistical/null-ensemble logic visible in the main text.
- Allow a slightly more technical figure-caption style than the Nature Physics-facing version.

## Shared locked language
- Use one contribution line throughout: false-positive-resistant nonlocal diagnosis of field-free topological superconductivity.
- Do not describe a local near-zero-energy mode as sufficient evidence.
- Use "combined diagnostic" or "evidence chain" rather than single-observable proof.
