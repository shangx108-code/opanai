# Figure resolution and SI separation upload specification

## Main manuscript
Upload main.pdf as the primary manuscript file.

## Supplementary Information
Upload supplementary_information.pdf separately from the main manuscript.

## Figures
- Prefer vector PDF
- Minimum raster fallback: 300 dpi
- Keep naming consistent (Fig2–Fig5)

## Source data and code
Archive source data and scripts in a DOI-minting repository.

## Caption discipline
Captions must specify what, how, and which false-positive route is excluded.
