# Archive Checklist

## Core manuscript package
- [ ] Title and abstract
- [ ] Main-text manuscript draft
- [ ] Supplementary note
- [ ] Verified bibliography file

## Theory package
- [ ] Unified Hamiltonian note
- [ ] Observable definitions and notation ledger
- [ ] Derivation note for scattering invariant and transport formulas

## Numerical package
- [ ] Reproducible code repository snapshot
- [ ] Parameter table for all main figures
- [ ] Raw data for main figures
- [ ] Raw data for supplementary figures
- [ ] Disorder-sampling metadata

## Figure package
- [ ] Figure 1 conceptual diagnosis figure
- [ ] Figure 2 spectra and wavefunctions
- [ ] Figure 3 failure of local tunnelling
- [ ] Figure 4 nonlocal/topological rescue
- [ ] Figure 5 diagnostic map / confusion matrix
- [ ] Final captions

## Review package
- [ ] Internal supervision notes
- [ ] Simulated referee reports
- [ ] Revision log by version
- [ ] Cover-letter draft

## Final submission package
- [ ] Submission PDF
- [ ] Source files
- [ ] Response-to-reviewers template
- [ ] Data/code availability statements
