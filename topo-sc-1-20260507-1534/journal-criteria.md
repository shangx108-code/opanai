# Journal Criteria

- Target journal: `Nature Communications`
- Current fit logic:
  - the project should read as a realistic, experimentally legible control framework for Majorana vortex physics
  - the paper should not rely on isolated zero-bias anomalies
  - novelty should come from deterministic programming of vortex topology through strain and dislocations
- Must-have evidence classes:
  - realistic Fe(Te,Se)-based model grounding
  - topology-sensitive diagnostic beyond plain low-energy peaks
  - vortex-state discrimination against trivial CdGM interpretations
  - experimentally readable LDOS or STM-style outputs in a verified regime
- Current not-yet-met thresholds:
  - no verified executable baseline model in the canonical root
  - no strain-controlled topological phase window yet returned by computation
  - no verified defect-vortex coupling result yet returned by computation
