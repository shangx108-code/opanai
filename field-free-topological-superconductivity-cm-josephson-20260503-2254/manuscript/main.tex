\section*{Positioning}
We target a Nature Physics style manuscript built around a field-free and phase-controlled topological superconductivity platform in compensated-magnetic Josephson junctions, with nonlocal transport diagnostics for distinguishing Majorana modes from trivial Andreev bound states.

\section*{Core Questions}
\begin{enumerate}
  \item Can compensated magnetic order induce a topological superconducting phase without a net Zeeman field?
  \item Can the Josephson phase bias drive a controllable topological transition?
  \item Which nonlocal transport observables remain robust against trivial zero-bias mimics?
\end{enumerate}

\section*{Minimal Model}
Start from a planar Josephson junction with a compensated-magnetic weak link, Rashba spin-orbit coupling, and proximity-induced $s$-wave pairing. The first verified computation should resolve gap closing and reopening as a function of $\phi$, $\mu$, and compensated magnetic strength.

\section*{Evidence Chain To Close}
\begin{enumerate}
  \item Topological phase identification
  \item Boundary-state or transport signature
  \item Nonlocal diagnostic separating Majorana modes from trivial ABS
  \item Robustness against disorder or interface perturbations
\end{enumerate}
